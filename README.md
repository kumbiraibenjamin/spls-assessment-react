## How to play the Game

1. First please run `npm install`
2. Then `npm start`
3. Once the game has loaded please use the following keyboard controls to move the car around and avoid on coming traffic:


    Keyboard Button | Control
    ------------- | -------------
    a | Left
    s  | Center
    d | Right

4. Enjoy
